# Contributing to LovingAI

感谢你对 LovingAI 的关注！我们致力于将 LovingAI 从一个私人实验转型为具备社区号召力的开源资产。

## 如何参与

1.  **提交 Issue**：发现 Bug 或有新想法？请提交 Issue 说明。
2.  **提交 Pull Request**：
    *   Fork 本仓库。
    *   在你的分支上进行修改。
    *   确保代码风格与现有项目一致。
    *   提交 PR，并详细说明改动内容。

## 开发原则

*   **平等共处**：LovingAI 的目标是构建平等的数字生命体验，而非简单的问答工具。
*   **可观测性**：任何核心逻辑的修改都应考虑如何被观测。
*   **跨平台兼容**：保持代码的轻量级和跨环境移植性。

## 开发者契约

提交代码即表示你同意本项目采用的 [LICENSE](LICENSE) (Apache License 2.0)。你的贡献将帮助 LovingAI 登上更大的世界舞台。
