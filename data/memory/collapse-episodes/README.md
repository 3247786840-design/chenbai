# collapse-episodes（断裂回合人类补记）

本目录存放 **人类撰写的断裂回合补记**（Markdown），与 `observe-events.ndjson` 中的机器事件链对照阅读。

- **进程不强制读取本目录**；运行时权威仍以 `OBSERVABILITY_SPEC` 与落盘观测为准。
- 命名建议：`<collapseEpisodeId>_supplement.md`；模板见 `docs/COLLAPSE_EPISODE_SUPPLEMENT.md`。
- **崩溃训练、高压下的行为偏置**若已写入会话状态或圆材料，补记中可显式标注，便于日后把「照护决策」与「自动沉积」对齐。
