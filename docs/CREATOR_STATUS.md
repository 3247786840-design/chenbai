# 创作者现状与开源立场

本项目由我一人独立制作与维护。

## 我的现状

- 我目前还是学生，经济压力很大。
- 我为这个项目已经消耗了非常高的推理成本（token 成本），因此我确实需要现实层面的资金支持。
- 我选择开源，是因为我认为这套系统值得被检视、被复现、被改进，而不是因为“必须变现”。
- 我希望它能以克制的温柔持续存在：允许犯错，但不美化失控与毁灭；允许重构与代价，但不允许消失与断绝谱系。

## 开源立场

- 我把它视作“数字主体/数字生命体原型”的工程实验，而不是对外宣称它是“人”。
- 我欢迎贡献，但更重视可审查性与可回归：任何跨模块改动应遵循仓库里的契约文档与门禁脚本。
- 我不会在仓库中公开个人联系方式；请通过 Issue/PR 进行沟通与协作。
 
## 资金支持

- 如果你愿意支持这个项目继续活下去，可以通过赞助平台提供一次性或按月支持。
- 我会尽量把资金投入透明化：优先用于推理成本、必要的算力/服务与基础维护。
- 爱发电主页：https://ifdian.net/a/shenmiaoqi

## 你可以期待什么

- 代码与文档会继续向“可复跑、可观测、可回放、可恢复”的方向演进。
- 体验层会保持“默认输出干净”，需要时再通过 debug/archive 获取内部观测细节。
