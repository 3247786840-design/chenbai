## 🔧 **编译与运行**

### **最简单方式（确保第一阶段运行）：**
```bash
# 1. 进入项目目录
cd <repo_root>\LovingAI

# 2. 编译核心文件
javac -d build\classes src\core\Circle.java src\core\EmotionCore.java src\LivingAI.java

# 3. 运行第一阶段
java -cp build\classes LivingAI
```

### **完整方式（包含第二阶段）：**
```bash
# 编译所有文件（可能会有依赖错误）
javac -d build\classes src\core\*.java src\*.java

# 运行完整系统
java -cp build\classes Launcher
```

### **如果第二阶段编译有问题：**
```bash
# 只运行第一阶段
java -cp build\classes Launcher --skip-phase2
```
